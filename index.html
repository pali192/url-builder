How to use the URLBuilder

1. Enter the URL (starting with 'http://' or 'https://').
2. Choose your desired Channel (some of the UTM parameters will be prefilled automatically).
3. The first three UTM parameters are mandatory and always needs to be filled out.
4. Dropdown fields which needs to be filled are highlighted (blue).
5. Other 5 UTM parameters are optional, so they should not be always filled out.
6. The greater the number of parameters, the better we can identify the campaign.
7. Click on the 'EXPORT URL' button to export your desired URL.
Your URLs will be stored in the URLs column in the table below.
(stored with time/date and optionally also with the separate utm parameters) It does not make any difference if your URL ist ending with or without a slash at the end.
If there is a highlighted dropdown, choose the value from the dropdown or fill the cell manually.
Use correct website URLs. https://www.example.com/, https://www.example.com/?parameter, https://www.example.com, https://www.example.com?parameter

We recommend that you always use lowercase letters, unless you have a specific purpose of using capital letters.
Keep your tags as short as possible, but keep them unique, easy to remember and understand.
Don't use a space, even when UTM parameters text seems to need one. Instead, separate words with plus +, underscore _ or dash-.
UTM tracking will not work as it should if it's tracking will look like broken links.
Other characters must be used to separate two or more words.
Ampersand already has a specific meaning when it comes to tracking links - it is the character that separates UTM parameters.
So it should not be used for anything else in that link. Remember, only the first UTM tag needs a question mark placed before it - other tags should be preceded by an ampersand.
‘#’ and ‘=’ cannot be used inside a UTM tag value. Don't use them.
UTM tags are case-sensitive in GA, so for example 'Example' will be recorded as a different source to 'example'.
